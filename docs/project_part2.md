# Project Part II

## Overview
Project Part II has two main graded components:

- **Presentation** — **10%**
- **Report** — **90%**

The presentation is scheduled for **the week of April 20**. Teams will present in **random order**, each presentation will be **about 8 minutes**, and there will be **Q&A**. The results **do not need to be final** because the goal is to get feedback. **All students must attend both sessions**, and **one or more students per project can present**. Students should use the **slide deck template on Canvas**. For the written report, students should use the **Word template on Canvas**. ## Detailed Requirements

### 1. Proposal (10%)
- Executive summary / abstract
- Clearly defined problem  [oai_citation:2‡Class 21 - Observational Studies Exercise.pdf](sediment://file_0000000090c871fd84d4beed873fa5a9)

### 2. Previous Work (15%)
- For the undergraduate version: related work from data sources  [oai_citation:3‡Class 21 - Observational Studies Exercise.pdf](sediment://file_0000000090c871fd84d4beed873fa5a9)

### 3. Data Description (15%)
Include:
- **Data collection**: how the data was collected or the source of the data
- **Data pre-processing**: transformations, merges, and processes performed
- **Data documentation**: explanation of each feature, including descriptions and ranges
- **Variables**: features (**independent variables**) and target/class/dependent variable  [oai_citation:4‡Class 21 - Observational Studies Exercise.pdf](sediment://file_0000000090c871fd84d4beed873fa5a9)

### 4. Modeling (15%)
Requirements depend on the task type:

#### Prediction task
- At least **3 machine learning algorithms**
- At least **1 model per algorithm**
- Algorithms and model parameters must be **described and justified**

#### Clustering task
- At least **3 clustering algorithms**
- At least **1 model per algorithm**
- Algorithms and model parameters must be **described and justified**

#### Estimation task
- At least **3 regression models** (linear or logistic)
- Models should use **different variables**
- Regression models and variables must be **described and justified**  [oai_citation:5‡Class 21 - Observational Studies Exercise.pdf](sediment://file_0000000090c871fd84d4beed873fa5a9)

### 5. Evaluation (10%)
Include:
- A clearly described **baseline**

Metrics depend on the task:

#### Prediction task
Use metrics discussed in class, such as:
- Precision
- Recall
- F1
- AUC-ROC

#### Clustering task
Use metrics discussed in class, such as:
- SSE
- Silhouette
- Purity

#### Estimation task
Use metrics discussed in class, such as:
- R²
- F-statistic
- AIC
- BIC  [oai_citation:6‡Class 21 - Observational Studies Exercise.pdf](sediment://file_0000000090c871fd84d4beed873fa5a9)

- Analyze and explain results
